Dự án “Xây dựng website bán sách truyện trực tuyến sử dụng ngôn ngữ lập trình Python” được lựa chọn nhằm xây dựng một hệ thống web hỗ trợ việc giới thiệu, quản lý và kinh doanh sách truyện trên môi trường Internet. Hệ thống cho phép người dùng tìm kiếm, xem thông tin sách, đặt mua trực tuyến, đồng thời hỗ trợ quản trị viên quản lý toàn bộ dữ liệu và hoạt động của website một cách hiệu quả.
Mục tiêu của dự án
-	Dự án hướng tới các mục tiêu chính sau:
-	Xây dựng một website bán sách truyện trực tuyến hoạt động ổn định, có giao diện thân thiện và dễ sử dụng.
-	Áp dụng ngôn ngữ lập trình Python cùng framework web (Django/Flask/FastAPI) để phát triển hệ thống theo mô hình chuẩn.
-	Thiết kế và triển khai cơ sở dữ liệu phục vụ việc lưu trữ và quản lý thông tin người dùng, sách, danh mục, đơn hàng.
-	Thực hiện đầy đủ các chức năng nghiệp vụ cơ bản của một website bán hàng trực tuyến như: đăng ký, đăng nhập, phân quyền, giỏ hàng, đặt hàng và thống kê báo cáo.
-	Rèn luyện kỹ năng phân tích yêu cầu, thiết kế hệ thống và triển khai phần mềm thực tế cho sinh viên.
